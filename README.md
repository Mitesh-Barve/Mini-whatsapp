# Mini-Whatsapp

A simple, lightweight chat application built using **MongoDB, Express.js, and Node.js**.

## 🚀 Features
- Real-time messaging  
- User authentication  
- Group chat support
- 
## 🛠️ Tech Stack
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB
